# anonfiles.js
 anonfiles.js utilizes the AnonFiles API to allow users to upload and request data about files.

# Installation
```bash
npm i anonfiles.js
```

# Usage
```javascript
const AnonFiles = require("anonfiles.js");
const API = new AnonFiles();
```
